# STVGP — генетическое программирование торговых стратегий

Самодостаточный модуль: коэволюция трёх популяций (long / short / meta)
через DEAP. Перенесён в отдельную папку — основной проект развивается
в сторону funding-arbitrage, а STVGP остаётся как исследовательская ветка,
которую можно запустить в любой момент.

## Запуск

```bash
cd project/stvgp
python main.py
```

Меню:
1. Новая эволюция (обучение, результат → `evolution_results.json`)
2. Загрузить готовую стратегию из файла + графики/метрики
3. Торговля на демо-балансе Binance Futures (нужны ключи)
4. Paper-trading на живых ценах без ключей

## Файлы

- `config.py` — параметры (символ, окно, периоды обучения/валидации)
- `data_loader.py` — загрузка свечей с Binance, индикаторы
- `primitives.py` — GP-примитивы (операции + тех. индикаторы)
- `stvgp.py` — движок эволюции (DEAP toolbox, evolve)
- `signals.py` — симулятор, fitness-функции, метрики, графики
- `storage.py` — сохранение/загрузка стратегий (JSON + filelock)
- `live_trading.py` — демо-торговля и paper-trading
- `main.py` — точка входа с меню
- `evolution_results.json` — сохранённые стратегии

## Важно про результаты

GP-стратегии склонны к переобучению: высокий train-fitness НЕ означает
прибыль на новых данных. На OOS-валидации лучшие стратегии давали ~0%
против +35% у buy & hold. Это исследовательский, а не боевой инструмент.
